El DOM: nuestro lugar de trabajo en la web
En programación existen objetos, estos son como envolturas para código, el navegador tiene algunos nativos cómo:

navigator: El objeto que contiene las funciones del navegador, también te permite acceder también al sistema operativo como el gps,
 guardar datos en el disco duro, etc.

window: El objeto que maneja cada una de las pestañas.

document: El objeto que contiene todo lo que vemos dentro de nuestra página

En aplicaciones web tenemos un concepto llamado DOM (Document Object Model) es la forma en que internamente el navegador organiza todo
el HTML dentro de una estructura de árbol
